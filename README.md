# 便签工具 (Sticky Notes)

基于 Electron 的多窗口桌面便签应用。每张便签一个独立窗口，像真实便签纸一样自由摆放在桌面上。

## 功能特性

- **多窗口架构** — 每张便签一个独立无边框窗口，自由拖拽定位，重启自动恢复位置和尺寸
- **Markdown 编辑** — 支持 Markdown 语法，右键预览历史版本中的渲染效果
- **多彩便签** — 8 种柔和配色（淡黄/淡粉/淡蓝/淡绿/淡紫/淡橙/白色/浅灰），右键即时切换
- **标签系统** — 便签底部标签徽章，自动补全已有标签，控制面板按标签筛选
- **全文搜索** — 控制面板搜索框实时过滤（匹配标题/内容/标签），250ms 防抖
- **置顶管理** — 便签窗口可置顶（始终保持在最前），标题栏图钉图标指示状态
- **归档功能** — 暂不需要的便签可归档，随时从控制面板恢复
- **历史版本** — 每次自动保存生成快照（最多 10 个），支持预览和回滚
- **深色模式** — 亮色 / 暗色 / 自动跟随系统，三态循环切换
- **自动保存** — 编辑后 1.5 秒防抖自动写入 localStorage
- **跨窗口同步** — 多个便签窗口 + 控制面板之间通过 storage 事件实时同步
- **导入 / 导出** — JSON 格式备份恢复全部数据；支持导出单张/全部 TXT
- **主控制面板** — 集中管理所有便签：新建、搜索、归档切换、导入导出

## 技术栈

| 层面 | 技术 |
|------|------|
| 桌面框架 | Electron 28 |
| 前端 | 原生 HTML / CSS / JavaScript，零运行时依赖 |
| 存储 | 浏览器 localStorage（JSON 序列化） |
| 窗口状态持久化 | electron-store（JSON 文件） |
| Markdown | 自研解析器，XSS 安全 |
| 打包 | electron-builder（NSIS 安装包） |

## 项目结构

```
sticky-notes-app/
├── main.js              # Electron 主进程：窗口管理、IPC 路由、Store
├── preload.js           # 预加载脚本：contextBridge 暴露安全 API
├── index.html           # 单便签视图（?noteId=xxx 加载指定便签）
├── control.html         # 主控制面板（便签列表、搜索、导入导出）
├── package.json         # 项目配置、依赖与构建脚本
├── README.md            # 本文件
└── release/             # 构建输出（已加入 .gitignore）
```

### 文件职责

- **main.js** — 应用入口。创建/管理 BrowserWindow 池（noteWindows 映射表），IPC 通信（创建/关闭/置顶/位置同步），electron-store 持久化窗口位置和尺寸。
- **preload.js** — 安全桥接。通过 `contextBridge` 向渲染进程暴露 `window.electronAPI`，包含创建窗口、关闭窗口、切换置顶、同步位置等接口。
- **index.html** — 单便签视图。通过地址栏参数 `?noteId=xxx` 加载指定便签，提供编辑、标签、颜色、版本历史、右键菜单、导出纯文本等功能。
- **control.html** — 控制面板。便签列表管理：新建、搜索、活跃/归档切换、导入导出、深色模式切换。

## 安装与运行

### 前提

- [Node.js](https://nodejs.org/) >= 18.x（推荐 20 LTS）
- npm >= 9.x

### 步骤

```bash
# 进入项目目录
cd sticky-notes-app

# 安装依赖
npm install

# 启动应用
npm start
```

### 打包为 Windows 安装包

```bash
npm run build
```

构建完成后，安装包输出在 `release/` 目录（NSIS `.exe` 安装程序）。

> 免安装版：`npm run build:dir`，输出在 `release/win-unpacked/`。

## 使用说明

### 创建便签

- 点击控制面板的 **＋ 新建** 按钮
- 快捷键 **Ctrl+N**（在控制面板窗口按下）

### 编辑便签

- 便签窗口标题栏可编辑标题（点击标题文字）
- 正文区支持 Markdown 纯文本输入
- 编辑内容自动保存（1.5 秒无操作后保存）

### 窗口操作

- 拖拽标题栏移动窗口
- 拖拽窗口边框/右下角调整大小（220×200 ~ 700×800）
- 关闭窗口（×）→ 确认删除便签

### 右键菜单

在便签窗口任意位置右键，可操作：

| 菜单项 | 功能 |
|--------|------|
| 🎨 切换颜色 | 8 色色块选择 |
| 📌 置顶/取消置顶 | 窗口置顶切换 |
| 📦 归档/恢复 | 归档便签（窗口关闭，数据保留） |
| 🏷️ 管理标签 | 标签徽章 + 输入 + 自动补全 |
| 🕒 历史版本 | 预览和回滚之前的快照 |
| 📄 导出 TXT | 导出当前便签为 .txt 文件 |
| 🗑️ 删除 | 确认后删除便签并关闭窗口 |

### 快捷键

| 快捷键 | 作用 | 适用窗口 |
|--------|------|----------|
| Ctrl+N | 新建便签 | 控制面板 |
| Ctrl+F | 聚焦搜索框 | 控制面板 |
| Ctrl+S | 手动保存 | 便签窗口 |
| Ctrl+Shift+D | 删除当前便签 | 便签窗口 |
| Escape | 关闭弹窗/菜单 | 全部 |

### 控制面板

- **活跃/归档标签页** — 切换显示活跃便签或已归档便签
- **搜索框** — 输入关键词实时过滤列表
- **点击列表项** — 打开/聚焦对应的便签窗口
- **右键列表项** — 打开、归档/恢复、删除
- **底部按钮** — 导出全部 TXT、导入 JSON、导出 JSON

## 数据存储

所有便签数据存储在浏览器 localStorage 中（键 `sticky_notes_data`），JSON 格式：

```json
{
  "version": 2,
  "notes": [
    {
      "id": "ljx123abc",
      "title": "",
      "content": "# 购物清单\n- 牛奶\n- 面包",
      "tags": ["购物"],
      "isPinned": false,
      "isArchived": false,
      "x": 100,
      "y": 200,
      "width": 320,
      "height": 360,
      "color": "#fff9c4",
      "created": 1705000000000,
      "updated": 1705000000000,
      "versions": []
    }
  ],
  "settings": {
    "darkMode": false,
    "themeAuto": true
  }
}
```

窗口位置和尺寸由 `electron-store` 持久化在用户数据目录（`%APPDATA%/sticky-notes/window-state.json`）。

## 数据备份

- **导出 JSON**：控制面板 → 📤 按钮，下载完整备份文件
- **导出 TXT**：便签窗口右键 → 📄，下载单张纯文本
- **导出全部 TXT**：控制面板 → 📄 按钮，合并所有活跃便签
- **导入 JSON**：控制面板 → 📥 按钮，选择之前导出的备份文件

> 建议定期导出 JSON 备份，避免因浏览器缓存清理等原因丢失数据。
